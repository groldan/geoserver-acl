# Images for Developer's Guide

This directory is for images used in the GeoServer ACL Developer's Guide documentation.

Images should include:

- Architecture diagrams
- Class diagrams
- Sequence diagrams
- Screenshots of development tools and processes

## Guidelines for Images

- Use SVG format for diagrams when possible
- Use PNG with reasonable resolution for screenshots
- Keep file sizes optimized for web
- Use clear, descriptive filenames
- Add meaningful alt text in documentation references
