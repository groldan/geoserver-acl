# GeoServer ACL Documentation Images

This directory contains images used in the GeoServer ACL documentation, specifically in the user's guide.

Images should be:
- Web-optimized (PNG or SVG preferred)
- Named descriptively
- Referenced correctly from AsciiDoc documents using `:imagesdir: images` attribute

When adding new screenshots or diagrams:
1. Use clear, high-contrast visuals
2. Crop to relevant content
3. Use annotations where helpful
4. Optimize for file size
